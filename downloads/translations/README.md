# FABIAN — Переводы для лендинга

Каталог содержит переводы 5 ключевых документов FABIAN на три языка — для отдачи с лендинга.

## Структура

```
downloads/translations/
├── EN/    — английские переводы (5 файлов + README)
├── FR/    — французские переводы (5 файлов + README, статус: working draft)
└── AR/    — арабские переводы (5 файлов + README, статус: working draft)
```

## Источник правды

Канонические версии переводов находятся в:
`E:\Проекты Аслана\Fabian Water HoReCa СНГ\13_Переводы\`

Файлы в `downloads/translations/` — копии для веб-раздачи. При обновлении канонических версий нужно перекопировать их сюда (см. инструкцию в основном README репозитория переводов).

## Как добавить переводы лендинга, если их нужно расширить

1. **Новый язык** (например, китайский):
   - Создать `13_Переводы/ZH/` и `downloads/translations/ZH/`.
   - Перевести 5 базовых документов.
   - Добавить README в `ZH/`.
   - Обновить главный README в `13_Переводы/` (добавить колонку).
   - Обновить главный лендинг `index.html` — добавить блок ссылок ZH в секцию загрузок.

2. **Новый документ** в существующем языке:
   - Перевести в `13_Переводы/{LANG}/`.
   - Скопировать в `downloads/translations/{LANG}/`.
   - Добавить запись в `13_Переводы/{LANG}/README.md`.
   - Обновить ссылку на главном лендинге.
